
 Smart Exam and Assignment Management System ## Project Overview The main objective of this project is to help teachers reduce the time and effort required to manage, distribute, and grade assignments and exams. Instead of manually correcting every student's paper, teachers can upload assignments or exams to the platform, where students complete them online. The system automatically grades objective questions, provides instant feedback, and uses Artificial Intelligence (AI) to evaluate essay questions. The platform also serves as a complete classroom management system by handling attendance, student performance, reports, learning materials, and subscription management. --- # Key Features ### 1. Online Exams and Assignments * Teachers can create and publish exams, quizzes, worksheets, and assignments. * Students complete them online from any device. * The system automatically grades objective questions. ### 2. QR Code Access * Teachers can generate a QR code for each exam or assignment. * Students simply scan the QR code to open the exam instantly. ### 3. Progressive Web App (PWA) * The platform is developed as a Progressive Web App (PWA), allowing users to install it once and access it quickly with an app-like experience. ### 4. Teacher Dashboard * A centralized dashboard displays: * Exams * Assignments * Student records * Attendance * Grades * Performance statistics ### 5. Attendance Management * Teachers can record student attendance. * Attendance records are stored alongside grades for assignments, worksheets, quizzes, and exams. ### 6. Student Performance Reports * All assignments, worksheets, quizzes, and exam results for each class are organized into a single report. * Reports can be shared with parents to monitor their child's academic progress. ### 7. Instant Feedback * After submitting an exam or assignment, students can: * View their score. * See their incorrect answers. * Read teacher comments for each question. * Access additional learning resources such as YouTube videos, PDFs, or external links. ### 8. Time Tracking * The system records: * Total exam completion time. * The time spent answering each individual question. * Teachers can: * Set a maximum time limit for each exam. * Schedule when an exam becomes available and when it closes. ### 9. Teacher Subscription Management * An administrator dashboard allows system administrators to: * View every teacher's remaining subscription days. * Extend subscriptions individually. * Extend subscriptions for all teachers with a single action. ### 10. Remaining Subscription Display * When teachers log in using their username and ID, the dashboard displays the number of subscription days remaining. ### 11. Student Accounts * Every student has: * Full Name * Student ID * Password ### 12. Progress Analytics * The system calculates each student's improvement percentage based on: * The last 10 exams. * The last 10 assignments or worksheets. * Progress is displayed using visual charts and statistics. ### 13. Automatic Daily Backup * All platform data is automatically backed up every day to prevent data loss. ### 14. Learning Materials * Teachers can upload notes, worksheets, and study materials for students to download. ### 15. AI-Based Essay Grading * Artificial Intelligence is used to evaluate essay questions and provide suggested scores and feedback. ### 16. Automatic Student Code Generation * The platform can automatically generate unique IDs or access codes for students. ### 17. Student Credentials Management * Teachers can view each student's ID and password whenever needed. ### 18. Flexible Question Types * Teachers can create multiple-choice questions using: * A, B, C, D, E * أ، ب، ج، د، هـ * The number of answer choices for each question is fully customizable. ### 19. Queue-Based Result Processing * After submitting an exam, students see a message indicating that their submission is being processed. * The system displays the estimated waiting time before grading is completed. * This queue system helps maintain performance during periods of high traffic. --- # Technologies and Concepts * Progressive Web App (PWA) * QR Code Integration * Automatic Grading * AI-Based Essay Evaluation * Performance Analytics * Attendance Management * Daily Automated Backup * Queue Management System * Student & Teacher Management * Parent Reporting System --- # Expected Benefits * Significantly reduces teachers' grading workload. * Saves time through automatic grading. * Provides immediate feedback to students. * Tracks student performance over time. * Simplifies attendance management. * Improves communication with parents through detailed reports. * Ensures data security with daily backups. * Supports both objective and essay-based assessments. * Delivers a fast, mobile-friendly experience through PWA technology. * Remains reliable even under heavy usage through queue-based processing.
